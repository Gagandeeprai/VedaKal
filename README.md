# Panchanga App (Hindu Calendar) 🕉️

A premium, modern **Panchanga App** built with React Native and Expo. It provides real-time Vedic astrological calculations based on your location.

## ✨ Features

- **📍 Location-Based:** Accurate Tithi, Nakshatra, Yoga, and Karana calculations based on your specific city (Latitude/Longitude).
- **🏙️ 20+ Cities:** Built-in support for major Indian cities and Karnataka district headquarters.
- **🕒 Real-time:** Dynamic calculations using `astronomy-engine` for high precision.
- **🎨 Premium UI:** Glassmorphism design, smooth animations, and comprehensive **Dark Mode** support.
- **⚠️ Inauspicious Times:** Daily calculations for Rahu Kalam, Yamaganda, and Gulika Kalam.
- **⚙️ Customizable:** Change language (English, Kannada, Sanskrit), location, and theme via a centralized Settings panel.

## 🚀 Technologies

- **Framework:** React Native (Expo SDK 52)
- **Language:** TypeScript
- **Astronomy:** `astronomy-engine`
- **State Management:** `zustand`
- **Navigation:** Expo Router
- **UI:** SVG icons, custom glassmorphism components

## 🛠️ Get Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the app:**
   ```bash
   npx expo start
   ```

3. **Run on Device:**
   - Scan the QR code with **Expo Go** (Android/iOS).
   - Or press `a` for Android Emulator / `i` for iOS Simulator.

## 📱 How to Use

1. **Auto-Location:** The app defaults to **Mangaluru**.
2. **Change City:** Tap the **⚙️ Gear Icon** in the top right → Tap **Location** → Select your city.
3. **Change Language:** In Settings, toggle between English, Kannada, or Sanskrit.
4. **Dark Mode:** In Settings, switch Theme to Dark or Auto (System Default).
